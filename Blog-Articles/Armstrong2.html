Armstrong Number

By: Shifas Shajahan

Algorithm to check whether the given number is armstrong or not

  1. Start
  2. Declare Variable sum, temp, num
  3. Read num from User.
  4. Initialize Variable sum=0 and temp=num.
  5. Repeat Until num>=0
  6. sum=sum + cube of last digit i.e [(num%10)*(num%10)*(num%10)]
  7. num=num/10
  8. IF sum==temp Print "Armstrong Number" ELSE Print "Not Armstrong Number"
  9. Stop
