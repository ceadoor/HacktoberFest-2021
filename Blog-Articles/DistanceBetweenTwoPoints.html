Distance between Two Points

By: Ajay James

This algorithm is to find distance between two points.

  1. Start
  2. Declare variables x1, x2, y1, y2, d
  3. Read values for x1, x2, y1, y2
  4. Assign d value as [(x2-x1)*(x2-x1)+(y2-y1)*(y2-y1)]
  5. Then assign d=sqrt(d)
  6. Display d value
  7. Stop
